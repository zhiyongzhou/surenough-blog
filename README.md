# surenough blog  流程

## 安装环境

- PHP >= 7.2.5
- Laravel7.x
- BCMath PHP 拓展 
- Ctype PHP 拓展 
- Fileinfo PHP 拓展 
- JSON PHP 拓展 
- Mbstring PHP 拓展 
- OpenSSL PHP 拓展 
- PDO PHP 拓展 
- Tokenizer PHP 拓展 
- XML PHP 拓展





## 安装流程

### Laravel7使用前端脚手架

```shell
$ composer require laravel

$ php artisan ui bootstrap
$ php artisan ui vue

$ npm install

$ npm run dev
```
